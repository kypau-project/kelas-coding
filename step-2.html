Step 2 - Struktur Dasar

Membuat struktur dasar HTML untuk website Link Bio

ðŸ“¦ Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa untuk membuat struktur halaman web. Bayangkan HTML sebagai kerangka atau tulang dari website-mu.

Analogi: Kalau website adalah rumah, HTML adalah struktur bangunannya (dinding, pintu, jendela), CSS adalah cat dan dekorasinya, dan JavaScript adalah sistem listriknya.

ðŸ“„ Buat File index.html

Mari kita mulai dengan membuat file utama. Nama index.html adalah nama standar untuk halaman utama website.

Kode Dasar HTML:

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Link Bio Saya</title>
</head>
<body>

  <!-- Konten akan kita tambahkan di sini -->

</body>
</html>

ðŸ“– Penjelasan Setiap Bagian

<!DOCTYPE html>

Memberitahu browser bahwa ini adalah dokumen HTML5 (versi terbaru HTML).

<html lang="id">

Tag pembuka dokumen HTML. lang="id" menandakan bahasa Indonesia.

<head>

Bagian "kepala" yang berisi informasi tentang halaman:

  • <meta charset="UTF-8"> - Mendukung karakter khusus seperti emoji
  • <meta name="viewport"...> - Agar tampil bagus di HP
  • <title> - Judul yang muncul di tab browser

<body>

Bagian "badan" tempat semua konten yang terlihat di halaman.

ðŸ“¦ Tambahkan Container

Sekarang, mari tambahkan struktur dasar untuk konten Link Bio:

<body>
  <div class="container">
    <!-- Avatar akan di sini -->
    
    <!-- Nama dan Bio akan di sini -->
    
    <!-- Link-link akan di sini -->
  </div>
</body>

Kenapa pakai div?

<div> adalah elemen "wadah" generik yang bisa kita style dengan CSS nanti. Class container akan membantu kita mengatur tata letak.

ðŸŽ¨ Tambahkan CSS Dasar

Mari tambahkan sedikit styling dasar agar terlihat lebih rapi. Tambahkan ini di dalam <head>: 

<style>
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: Arial, sans-serif;
  background-color: #1a1a1a;
  color: white;
  min-height: 100vh;
  display: flex;
  justify-content: center;
  align-items: center;
}

.container {
  text-align: center;
  padding: 20px;
  max-width: 400px;
  width: 100%;
}
</style>
