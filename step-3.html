Step 3 - Avatar

Menambahkan foto profil ke website Link Bio

üé® Apa itu Avatar?

Avatar adalah gambar profil yang mewakili identitasmu di internet. Ini bisa berupa foto diri, ilustrasi, atau gambar apapun yang kamu suka!

Tips: Gunakan gambar dengan rasio 1:1 (persegi) agar terlihat rapi saat dibuat bulat.

üì∑ Siapkan Gambar Avatar

Kamu punya beberapa opsi untuk gambar avatar:

Opsi 1: Gunakan Foto Sendiri

  • Upload foto ke repository GitHub-mu
  • Atau gunakan layanan hosting gambar gratis

Opsi 2: Gunakan Placeholder

Untuk sementara, kita bisa pakai layanan placeholder seperti:

https://api.dicebear.com/7.x/avataaars/svg?seed=NamaKamu

Opsi 3: Gunakan UI Avatars

Generate avatar berdasarkan inisial nama:

https://ui-avatars.com/api/?name=Nama+Kamu&background=random

üñºÔ∏è Tambahkan Tag Image

Tambahkan kode ini di dalam <div class="container">:

<!-- Avatar -->
<img 
  src="https://api.dicebear.com/7.x/avataaars/svg?seed=Felix" 
  alt="Foto Profil" 
  class="avatar"
>

Penjelasan:

  • src - Alamat (URL) gambar
  • alt - Teks alternatif jika gambar tidak bisa dimuat
  • class - Nama class untuk styling CSS

‚ú® Styling Avatar dengan CSS

Tambahkan CSS ini untuk membuat avatar terlihat keren:

.avatar {
  width: 120px;
  height: 120px;
  border-radius: 50%;
  object-fit: cover;
  border: 4px solid #333;
  margin-bottom: 20px;
  transition: transform 0.3s ease;
}

.avatar:hover {
  transform: scale(1.05);
}

Penjelasan CSS:

  • width & height - Ukuran gambar 120x120 pixel
  • border-radius: 50% - Membuat gambar menjadi bulat
  • object-fit: cover - Gambar mengisi area tanpa distorsi
  • border - Garis tepi di sekeliling avatar
  • transition - Animasi smooth saat hover

üé≠ Variasi Gaya Avatar

Avatar dengan Bayangan:

.avatar {
  box-shadow: 0 4px 15px rgba(0, 0, 0, 0.3);
}

Avatar dengan Border Gradient:

.avatar-wrapper {
  padding: 4px;
  background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
  border-radius: 50%;
  display: inline-block;
}

.avatar {
  border: none;
}
